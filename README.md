# Montando um servidor SAMBA
O Samba é uma ferramenta de software livre que funciona como servidor para compartilhar arquivos, pastas e impressoras entre sistemas Linux/Unix e Windows, utilizando o protocolo SMB/CIFS. Ele permite a interoperabilidade em rede, atuando como servidor de arquivos, controlador de domínio e gerenciando permissões de acesso de forma segura.
